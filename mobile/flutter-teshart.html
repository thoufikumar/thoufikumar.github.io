Teshart

Art Order Managing App

What is Teshart?

Teshart helps artists manage custom orders, customers, and payments in one place.

What Techstacks are used?

Built using Flutter, OAuth authentication, Firestore database, and Google Scripts automation.

Why it matters?

Artists often manage orders, payments, and client requests across scattered platforms.

Future Enhancements ?

Future updates include smarter order analytics and an improved V2 management experience.

Core UI Flow

  • Order status tracking
  • Active order overview
  • Quick order access
  • Incoming request review
  • Order request details
  • Accept reject decisions
  • Order receipt tracking
  • Completed order records
  • Transaction history reference
  • Advance payment entry
  • Secure payment confirmation
  • Order payment validation
  • Client communication space
  • Order discussion thread
  • Quick response messaging
  • Artist account management
  • Notification preferences control
  • Performance insights access
  • Customer contact list
  • Client information access
  • Order customer management
